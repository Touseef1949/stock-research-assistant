# Changelog

All notable changes are documented here. The project follows
[Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-07-18

### Added

- reproducible runtime and development locks used by local, CI, and deployment
- one local quality command with formatting, linting, boundary type checks,
  deterministic tests, coverage, and research routing/grounding gates
- security policy, AI/financial-data threat model, contribution guidance, and
  pull-request template
- least-privilege immutable CI and dependency-update automation
- centralized model and prompt-policy version identifiers
- deterministic test mode that prevents refreshes from rewriting the tracked
  NSE symbol-master cache

### Documented

- supported Python/runtime, maturity, architecture, privacy/trading boundaries,
  limitations, evaluation layers, and release workflow
